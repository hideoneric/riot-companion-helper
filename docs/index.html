Features

What it does

Three things, done automatically.

Auto-Launch

Detects League of Legends and Valorant. Launches Blitz.gg the moment either game starts.

Smart Close

Blitz stays open while any game is running. Only closes when every game has stopped.

Zero Friction

Runs silently in the system tray. Set it up once and never think about it again.

Getting Started

Up and running in minutes

Three steps. No technical knowledge required.

01

Install the app

Run the setup wizard. No admin rights needed.

02

Set your Blitz path

Point the app to your Blitz.exe once in Settings.

03

Play your game

Launch League or Valorant. Blitz opens and closes automatically.
