What it does

Three things, done automatically.

Auto-Launch

Detects League of Legends and Valorant. Launches Blitz.gg the moment any game starts.

Smart Close

Blitz stays open while any game is running. Only closes when all games are stopped.

Zero Friction

Runs silently in the system tray. Set it up once and forget it's there.
